Create: Sensors
======
This simple addon adds a few different redstone components to be used with Create and Create: Aeronautics.

__This is my first ever minecraft mod, so report any bugs to me. Or feature requests, because I can probably do those.__

Thank you.

 <br>
 
## Impact Sensor
<img width="300" height="145" alt="Image" src="https://github.com/user-attachments/assets/8fbbef42-8e54-4113-bdd2-01b15185c600" /> <br>

Will trigger a redstone signal when it detects deceleration above the threshold, which can be set on the block. <br>

## Impact Fuze
<img width="300" height="145" alt="Image" src="https://github.com/user-attachments/assets/df1e61b6-b9dc-4548-b4e4-0b666f492aa1" /> <br>

Will detonate as soon as it detects deceleration above the threshold, also set on the block. <br>

## Proximity Sensor
<img width="300" height="145" alt="Image" src="https://github.com/user-attachments/assets/3ab63ada-efa1-4789-8a57-ef9afbd65b3c" /> <br>

Will trigger a redstone signal when it detects a nearby sublevel/contraption when within the set range. <br>
